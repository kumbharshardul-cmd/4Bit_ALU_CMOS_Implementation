# 4Bit_ALU_CMOS_Implementation
This project demonstrates the design and layout implementation of a 4-bit Arithmetic Logic Unit (ALU) using CMOS technology.
The work focuses on transistor-level design and physical layout, providing a practical understanding of VLSI design flow from schematic to layout.

The ALU performs essential arithmetic and logical operations, and its layout is designed and verified in Microwind with attention to area optimization, power efficiency, and correct CMOS logic implementation.

+++Supports operations:+++
-Arithmetic: Addition, Subtraction
-Logical: AND, OR, XOR, NOT
-Full custom layout design in Microwind
-Transistor-level implementation (PMOS & NMOS)
-Design rule check (DRC) compliant layout
-Functional verification through simulation
